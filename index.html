Resumen Profesional

Graduado en Grado Superior de Desarrollo de Aplicaciones Multiplataforma. Actualmente poseo un año de experiencia profesional desarrollando en tecnología Python + AWS (Lambdas, RDS, S3) y Salesforce, en el que he podido trabajar para el sector energético, implementando funcionalidades a partir de diseños técnicos, desarrollando, configurando y desplegando lambdas. Desarrollo de scripts y funciones serverless con Python. Uso de Postman para consumo e integración de APIs REST. Creación de microservicios con FastAPI.

Además, también tengo experiencia en tecnología Java y creación de microservicios con Spring boot MVC, haciendo uso de Hibernate + JPA para la relación de entidades con base de datos como MySQL.

En cuanto a manejo de bases de datos tengo un año de experiencia en MySQL, realizando conexiones, consultas e implementando entidades. También me manejo con soltura con PostgreSQL. He trabajado con Docker y Git para control de versiones.

Conocimientos básicos en otras tecnologías como Javascript, VBA, .NET que se pueden ver reflejados en mi CV.
